**IMPORTANT — Three distinct app states:**
- **Active**: Widget visible, monitoring active, sounds enabled — normal operation.
- **Hidden**: Widget hidden via "Show / Hide Widget", monitoring STILL ACTIVE in background (isActive remains true). User can hear sounds, hotkeys still work.
- **Power Off**: Widget hidden via "Power Off", monitoring STOPPED (isActive = false). No sounds, no permissions, no hotkeys. Menu bar icon greyed out. User must "Power On" to resume.

```swift
MenuBarExtra {
    // SECTION 1: Status
    Text("Sessions: \(monitor.sessionCount)")
    Text(monitor.autoAccept ? "YOLO MODE" : monitor.state.label)
        .foregroundColor(.secondary)

    Divider()

    // SECTION 2: Controls
    Toggle("YOLO Mode", isOn: $monitor.autoAccept)
        .keyboardShortcut("y")
    Toggle("Auto-Speak", isOn: $monitor.autoSpeak)
        .keyboardShortcut("k")
    Toggle("Sound Effects", isOn: $monitor.soundEnabled)
        .keyboardShortcut("s")
    Toggle("Notifications", isOn: $monitor.notificationsEnabled)
        .keyboardShortcut("n")

    Divider()

    // SECTION 3: App
    Menu("Theme") {
        Picker("Color", selection: $themeManager.currentThemeId) {
            ForEach(ThemeManager.themes) { theme in
                Text(theme.name).tag(theme.id)
            }
        }
        Divider()
        Toggle("Dark Mode", isOn: $themeManager.darkMode)
    }

    Button("Show / Hide Widget") {
        Self.toggleMainWindow()
    }
    .keyboardShortcut("h", modifiers: [.command, .shift])

    Button(monitor.isActive ? "Power Off" : "Power On") {
        monitor.isActive.toggle()
        if !monitor.isActive {
            Self.hideMainWindow()
        } else {
            Self.showMainWindow()
        }
    }
    .keyboardShortcut("p")

    if !AXIsProcessTrusted() {
        Button("Enable Global Hotkeys...") {
            let opts = [kAXTrustedCheckOptionPrompt.takeUnretainedValue() as String: true] as CFDictionary
            _ = AXIsProcessTrustedWithOptions(opts)
        }
    }

    Divider()

    Button("Quit Claumagotchi") { NSApp.terminate(nil) }
        .keyboardShortcut("q")
} label: {
    Image(nsImage: EggIcon.image(state: monitor.menuBarIconState))
}
.menuBarExtraStyle(.menu)
```

**Key distinction:** "Show / Hide Widget" calls `toggleMainWindow()` which only toggles window visibility — it does NOT touch `monitor.isActive`. Monitoring, sounds, and hotkeys continue in the background. "Power Off" sets `monitor.isActive = false` AND hides the window — this stops all monitoring, sounds, and permissions.

**Add `hideMainWindow` static method** (if not already present — check first). Add below `showMainWindow`:
```swift
static func hideMainWindow() {
    for window in NSApp.windows where window.identifier?.rawValue == "main" {
        window.orderOut(nil)
        return
    }
}
```
