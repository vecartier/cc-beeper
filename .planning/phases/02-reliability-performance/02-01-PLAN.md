Two changes in ClaudeMonitor.swift to address PERF-02 and REL-03:

**PERF-02 — Stop reading sessions.json on every event:**

Current updateAggregateState() (lines 249-267) reads sessions.json from disk on every call to prune stale sessions. This is called on every event (pre_tool, post_tool, stop, session_start, session_end).

Instead of reading from disk every time:

1. Add a private property to track when pruning last happened:
   ```swift
   private var lastPruneTime: Date = .distantPast
   ```

2. In updateAggregateState(), only read sessions.json if at least 30 seconds have elapsed since the last prune:
   ```swift
   private func updateAggregateState() {
       // Prune stale sessions at most every 30 seconds
       if Date().timeIntervalSince(lastPruneTime) > 30 {
           if let sessionsData = FileManager.default.contents(atPath: Self.ipcDir + "/sessions.json"),
              let sessions = try? JSONSerialization.jsonObject(with: sessionsData) as? [String: Any] {
               let activeIds = Set(sessions.keys)
               for key in sessionStates.keys where !activeIds.contains(key) {
                   sessionStates.removeValue(forKey: key)
               }
           }
           lastPruneTime = Date()
       }

       let values = sessionStates.values
       if values.contains(.needsYou) {
           state = .needsYou
       } else if values.contains(.thinking) {
           state = .thinking
       } else {
           state = .finished
       }
   }
   ```

3. Also prune immediately on "session_end" events — in processEvent's session_end case, after `sessionStates.removeValue(forKey: sid)`, reset lastPruneTime to .distantPast so the next updateAggregateState call does a fresh prune. This ensures accuracy when sessions actually end.

**REL-03 — Replace manual idle Timer with DispatchWorkItem:**

The current idleTimer (lines 64, 289-294) uses Timer.scheduledTimer which requires RunLoop and manual invalidation. Replace with a simpler DispatchWorkItem pattern:

1. Change the property from `private var idleTimer: Timer?` to `private var idleWork: DispatchWorkItem?`

2. Replace startIdleTimer:
   ```swift
   private func startIdleTimer(interval: TimeInterval) {
       idleWork?.cancel()
       let work = DispatchWorkItem { [weak self] in
           guard let self, self.pendingPermission == nil else { return }
           self.state = .finished
       }
       idleWork = work
       DispatchQueue.main.asyncAfter(deadline: .now() + interval, execute: work)
   }
   ```

3. Replace all `idleTimer?.invalidate()` calls with `idleWork?.cancel()` (in processEvent and goToConversation)

4. In deinit, replace `idleTimer?.invalidate()` with `idleWork?.cancel()`

Do NOT change any permission logic, file watcher logic (that's Task 1), or state machine branching.
