TTSService should be a final class (NOT ObservableObject — ClaudeMonitor will own the state):

```
final class TTSService: @unchecked Sendable {
    private let synthesizer = AVSpeechSynthesizer()
    private var speechDelegate: SpeechDelegate?
    @Published var isSpeaking: Bool = false

    func speakSummary(_ text: String) async -> String
    func stopSpeaking()
}
```

Actually, make it an ObservableObject so isSpeaking can be observed. Keep it simple.

**speakSummary(_ text: String) async -> String:**
1. If text.count < 200, skip summarization, use text directly
2. Otherwise, check `SystemLanguageModel.default.availability`
3. If `.available`, create `LanguageModelSession` with system prompt (copy exact prompt from LoopEngine):
   ```
   You are reading an AI coding assistant's response.
   Extract ONLY the final conclusion — what was done and what the user should do next.
   Say it in first person as the assistant. Keep it to 1-3 short sentences.
   Skip all code, file paths, commands, and technical details.
   ```
4. Call `session.respond(to: String(text.suffix(2000)))`, return content
5. Fallback: return lastParagraph (split by "\n\n", filter empty/short, take last)
6. After summarization, call internal `speak()` with the result
7. Return the summary text (so caller can update lastSummary on screen)

**speak(_ text: String):**
- If already speaking, stop first
- Create AVSpeechUtterance with text
- Voice: `AVSpeechSynthesisVoice(identifier: "com.apple.voice.premium.en-US.Ava")`, fallback `AVSpeechSynthesisVoice(language: "en-US")`
- Rate: `AVSpeechUtteranceDefaultSpeechRate * 0.9`, pitch: 1.05
- Set speechDelegate (SpeechDelegate from LoopEngine pattern — onFinish sets isSpeaking = false)
- Set isSpeaking = true, synthesizer.speak(utterance)

**stopSpeaking():**
- synthesizer.stopSpeaking(at: .immediate)
- isSpeaking = false
- usleep(100_000) — give audio session time to release

**SpeechDelegate:** Copy from LoopEngine — NSObject, AVSpeechSynthesizerDelegate, calls onFinish on didFinish and didCancel.

**Logging:** Use same pattern as VoiceService — write to `/tmp/claumagotchi-tts.log`

Import: Foundation, AVFoundation, FoundationModels
